# Vegas Carpet Quiz

Can you identify the Las Vegas hotel by its carpet? Test your knowledge of iconic casino floor patterns from the Las Vegas Strip.

**[Play the Quiz](https://vegas-carpet-quiz.streamlit.app/)**

## Features

- **Two Quiz Modes**: Multiple choice or free text input
- **20 Iconic Carpets**: From Bellagio to Circus Circus
- **Score Tracking**: See how many you get right
- **Instant Feedback**: Learn the correct answer when you guess wrong

## Run Locally

```bash
uv venv && uv pip install streamlit
.venv/bin/streamlit run app.py
```

## Hotels Featured

Wynn, Park MGM, Mandalay Bay, Venetian, Mirage, Paris, Treasure Island, Caesar's Palace, Cosmopolitan, Resorts World, Excalibur, The Linq, Bellagio, Luxor, Planet Hollywood, New York New York, Circus Circus, Aria, Harrah's, and MGM Grand.
